## Project Overview
This project analyzes workforce data to identify key drivers of employee turnover. By connecting department volume, monthly income, tenure, and job satisfaction metrics, it provides data-driven strategies to help HR mitigate talent loss.

---


* **SQL:** Data cleaning, filtering (`Attrition = 'Yes'`), and aggregation in PostgreSQL via DBeaver.
* **BI / Visualization:** Tableau Public for interactive executive dashboard design.
* **Version Control:** Git & GitHub.

---

## Key Business Recommendations
* **Targeted Pay Adjustments:** Reevaluate salary structures for roles where high turnover volumes cross-reference with below-average monthly income.
* **Early Retention Interventions:** Deploy proactive HR touchpoints for specific roles approaching historical turnover timeline cliffs.
* **Morale Management:** Prioritize department reviews where job satisfaction scores drop below critical baselines.
